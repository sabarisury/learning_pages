Era 1 · Encoder
BERT: Deep Bidirectionality
2018

BERT marked a major inflection point by introducing full bidirectional context using the Transformer encoder stack. Unlike earlier models that processed text left-to-right or right-to-left, BERT looked at both directions simultaneously, enabling a richer contextual understanding of language.

Training Objectives

Masked Language Modeling (MLM) Next Sentence Prediction (NSP)

MLM selects 15% of tokens — 80% are replaced with [MASK], 10% with random tokens, 10% unchanged. NSP trains the model to classify whether sentence B actually follows sentence A.

Dataset

BooksCorpus ~800M words English Wikipedia ~2.5B words
Why it mattered: Solved the context bottleneck. Exceptional at semantic understanding, word disambiguation (e.g., "bank"), and downstream NLU tasks (GLUE benchmark).
RoBERTa follow-up (2019): + added Facebook AI's RoBERTa showed that NSP was actually harmful — removing it and training longer on more data substantially improved BERT's performance. This validated that MLM alone was the core driver of BERT's strength.
Era 2 · Decoder
GPT-1: The Decoder Seed
2018

While BERT focused on understanding, GPT-1 pivoted toward generation, using a decoder-only Transformer with causal (left-to-right) masking. It introduced the pre-train → fine-tune paradigm that would define the next several years.

Predict the next token given all previous tokens. Simple, yet powerful.

Key design choices: Causal masking (unidirectional), weight tying between input and output embeddings, post-norm architecture.
Era 3 · Scaling
GPT-2: Scaling + Pre-Normalization
2019

GPT-2 demonstrated that scale + better data = emergent intelligence. It was one of the first models OpenAI deliberately withheld from full release, citing misuse concerns — itself a cultural milestone.

Key Architectural Improvements

Pre-LayerNorm Residual Scaling 1/√N Larger context

Moving normalization before each sub-layer (pre-norm) dramatically improved gradient stability during training at scale. Residual streams were scaled by 1/√N to prevent blow-up as depth increased.

Dataset

WebText ~40GB (curated Reddit-linked data)
Breakthrough: Zero-shot learning — prompt alone could guide task behavior without any fine-tuning. Pre-norm became the standard for all subsequent large Transformer scaling.
Era 4 · Generalization
GPT-3: Few-Shot Learning
2020

GPT-3 scaled massively and introduced in-context learning as a core paradigm — a model that adapts to tasks simply by reading examples in the prompt, with no gradient updates.

Sparse Attention

Dense Attention (global) Sparse / Local Attention

Alternating dense and sparse attention layers reduced the quadratic O(n²) attention complexity, enabling longer sequences to be handled more efficiently at this parameter scale.

Dataset

Filtered Common Crawl WebText2 Books · Wikipedia
Paradigm shift: From fine-tuning → prompting. You no longer needed to retrain a model for each task — you could describe the task in natural language.

Mixture of Experts

In MoE, only a subset of the network (the "experts") activates for each token. This allows massive total model capacity while keeping per-token compute manageable — a key breakthrough for efficient scaling beyond 100B parameters.

Alignment Techniques

RLHF (PPO) Reward Model KL Penalty Constitutional AI / RLAIF
Constitutional AI (RLAIF): + added Anthropic pioneered a complementary alignment technique where an AI model critiques and revises its own outputs against a set of principles ("a constitution"), reducing reliance on large-scale human labeling. RLAIF (RL from AI Feedback) is now widely studied alongside RLHF across the industry.
Why it mattered: Solved the knowledge vs. compute tradeoff — massive capacity without proportionally massive inference cost. Reliability and safety improved significantly over GPT-3.
Era 6 · Reasoning + Agency
GPT-5: System 2 Reasoning & Agentic Intelligence
2025

GPT-5 represents a fundamental shift — from static next-token prediction to dynamic reasoning systems that allocate compute at inference time. The model can think before it responds.

Core Innovation: Inference-Time Compute Scaling

Chain-of-Thought (internal) Tree-of-Thought Search Self-Correction

Rather than spending all compute during training, the model allocates reasoning steps dynamically at inference. This enables verification, backtracking, and pruning of candidate reasoning chains — analogous to Kahneman's "System 2" deliberate thinking.

Broader Landscape
Beyond OpenAI: The Full Ecosystem
2019 – 2025

+ added The BERT→GPT lineage is the most studied, but the LLM landscape evolved across multiple parallel tracks. Understanding these is essential for practitioners.

Key Parallel Developments

T5 (Google, 2019): Reframed every NLP task as a text-to-text problem ("translate English to French: ..."), unifying the field under a single encoder-decoder architecture and training objective.
PaLM / Gemini (Google, 2022–2024): PaLM introduced breakthrough few-shot performance via chain-of-thought prompting at scale. Gemini 1.5 Pro introduced the 1M-token context window using a sparse Mixture-of-Experts architecture — the actual origin of million-token context.
LLaMA (Meta, 2023–2025): Open-weight models that proved you could match frontier performance at 7B–70B parameter scales with better data curation. Transformed the open-source ecosystem.
Claude (Anthropic, 2023–2025): Pioneer of Constitutional AI (RLAIF) — alignment without purely human labeling. Claude 3 Opus matched GPT-4 on major benchmarks. Introduced long-context windows (200K tokens) for document-level reasoning earlier than most competitors.

Key Takeaways for Practitioners

01 · Scaling
Scaling Has Evolved
Then: parameters + data. Now: inference-time compute + reasoning loops. More thinking at test time often beats bigger weights.
02 · Architecture
Architecture Choices Matter
Pre-norm enables stable scaling. MoE enables efficient capacity. Reasoning loops enable better decisions. Each solved a real bottleneck.
03 · Data
Data Strategy Shifted
Curated → web-scale → synthetic reasoning data. The frontier is now dominated by high-quality synthetic traces for reasoning tasks.
04 · Prompting
Prompting Is Programming
Few-shot examples, chain-of-thought, tool chaining, structured XML — prompt engineering is now a first-class engineering discipline.
05 · Agency
Models Are Agents
Not just answering: planning, tool use, executing, correcting. System design — not just model selection — is the core challenge now.
06 · Ecosystem
OpenAI ≠ Full Picture
Claude, Gemini, LLaMA, Mistral — the ecosystem is plural. Each lineage contributes distinct ideas: RLAIF, MoE, open weights, long context.
"From pattern recognition → language generation → reasoning systems → autonomous agents"

We are no longer just building models that predict text. We are building systems that can think, act, and collaborate. The next wave isn't just smarter models — it's smarter systems built on top of them.
